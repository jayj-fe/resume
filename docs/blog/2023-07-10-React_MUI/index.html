MUI란

MUI(Material UI)는 Material Design을 구현해놓은 라이브러리이다.
이 라이브러리를 이용하면 material 디자인 스타일이 적용된 UI를 매우 쉽게 구현할 수 있다.

Material Design이란?

구글은 모바일과 데스크탑 그리고 그 외 다양한 디바이스들을 아우르는 하나의 일관된 디자인 가이드라인을 공개하였는데, 이것이 바로 Material Design이다.

머티리얼 디자인은 질감이 느껴지는 표면 (tactile surfaces) 과 대담하고 선명한 그래픽 디자인 (bold graphic design), 그리고 아름답고 직관적인 사용자 경험을 위한 자연스러운 애니메이션을 특징으로 한다.


MUI 설치하기

MUI는 npm, yarn 등으로 설치할 수 있다.

// npm
npm install @mui/material @emotion/react @emotion/styled
// yarn
yarn add @mui/material @emotion/react @emotion/styled


MUI 사용하기

MUI 공식 페이지에서 원하는 컴포넌트를 찾고 사용하고자하는 위치에서 import를 하면 된다.

import { Button } from '@mui/material';

function App() {
return (
<>
<Button variant="text">Text</Button>
      </>
    );
}


export default App;


MUI 커스텀하기

sx라는 속성을 통해 인라인 스타일을 정의할 수도 있다.

import { Button } from '@mui/material';

function App() {
return (
<>
<Button
variant="text"
sx=
>
Text</Button>
      </>
    );
}


export default App;

기존의 CSS를 이용하여 스타일을 변경하는 방법도 있다.
