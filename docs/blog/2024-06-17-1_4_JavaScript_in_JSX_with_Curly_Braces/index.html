따옴표로 문자열 전달하기


  1. 문자열 어트리뷰트를 JSX에 전달하려면 작은따옴표나 큰따옴표로 묶어야 한다.
export default function Avatar() {
  return (
    <img
      className="avatar"
      src="https://i.imgur.com/7vQD0fPs.jpg"
      alt="Gregorio Y. Zara"
    />
  );
}


  1. 어트리뷰트를 JSX에 동적으로 전달하려면 중괄호로 묶어야 한다.
export default function Avatar() {
  const avatar = 'https://i.imgur.com/7vQD0fPs.jpg';
  const description = 'Gregorio Y. Zara';
  return (
    <img
      className="avatar"
      src={avatar}
      alt={description}
    />
  );
}

export default function TodoList() {
  const name = 'Gregorio Y. Zara';

  );

const today = new Date();

function formatDate(date) {
return new Intl.DateTimeFormat(
'en-US',
{ weekday: 'long' }
).format(date);
}

return (
<h1>To Do List for {formatDate(today)}</h1>
  );

”이중 중괄호” 사용하기: JSX의 CSS와 다른 객체

JSX에는 문자열, 숫자 및 기타 JavaScript 표현식뿐만 아니라 객체를 전달할 수도 있다.
또한 객체는 { name: “Hedy Lamarr”, inventions: 5 }처럼 중괄호로 표시된다.
따라서 JSX에서 객체를 전달하려면 person=와 같이 다른 중괄호 쌍으로 객체를 감싸야 한다.

export default function TodoList() {
  return (
    <ul style=>
      <li>Improve the videophone</li>
      <li>Prepare aeronautics lectures</li>
      <li>Work on the alcohol-fuelled engine</li>
    </ul>
  );
}

는 특별한 문법이 아니다. JSX 중괄호 안에 들어 있는 JavaScript 객체이다.

⚠️주의

인라인 style 프로퍼티는 캐멀 케이스로 작성된다.
예를 들어 HTML에서의 <ul style="background-color: black">은 컴포넌트에서 <ul style=>로 작성된다.
