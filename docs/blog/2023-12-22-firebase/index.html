FireBase - 데이터베이스 이용하기


FireBase 란?

FireBase는 2014년도에 구글에서 인수한 모바일, 웹 애플리케이션 개발 플랫폼이다. Firebase는 데이터베이스, 인증, 스토리지, 원격 구성, 푸시 알람 등다양한 기능을 제공한다.


FireBase 시작하기

  1. 공식사이트에 일단 가입해야한다.

  2. 공식사이트에서 로그인 후 console로 이동한다.

  3. 새 프로젝트를 추가한다.

  4. 프로젝트 이름을 설정한다.

  5. 계정을 선택한다.

  6. 생성이 완료되면 왼쪽 사이드 바에서 빌드 - Firestore Database를 선택한다.

  7. 데이터베이스를 생성한다.

  8. 안내에 따라 만든다. 위치는 빈 칸이어도 된다.

  9. 프로덕션 모드로 생성한다.

  10. 생성이 완료되면 컬렉션을 만든다.

  11. 문서 ID 필드를 설정해준다.

  12. 저장하면 생성된 것을 확인할 수 있다.


FireBase 사용하기

FireBase 연동

  1. 데이터베이스가 생성이 됐다면 상단에 프로젝트 설정으로 들어간다.

  2. 내 앱에서 앱을 생성한다.

  3. 앱 닉네임을 작성하고 저장한다.

  4. 코드가 생성된 걸 확인하고 콘솔로 이동한다.

  5. 프로젝트에서 npm을 통해 firebase를 설치한다.
    npm install firebase
  6. fiebase-config.js를 만들어준다.
