async 와 await 란

async는 asynchronous의 약자로 비동기 방식이다. async는 예전부터 많이 봤었다.

스크립트를 호출할 때도 사용했었으며,

<script type="text/javascript" src="common/js/script2.js" async></script>

스크립트는 비동기적으로 호출한다.
‘Where do you place the JavaScript?’ 포스터

Ajax 또한 Ajax(Asynchronous JavaScript and XML)의 약자로, ajax 옵션중 async옵션이 존재한다.

$.ajax({
    ...
    async: false,
    success: function(data) {

<span class="p">}</span>



});

async : false => ajax 호출시 비동기를 사용하지 않겠다는 설정

즉, async는 비동기함수를 만들어주는 명령어이다.
await는 은 (~을) 기다리다 라는 뜻으로, 비동기 방식인 함수를 해당 명령어로 통해 기다리게 만든다.


async 사용법

async와 await는 둘 다 사용법이 간단하다.
사용하고자 하는 함수의 앞에 async를 붙인다.

const asyncAPI = async () => {
    ...
}

async function() => {
...
}


async () => {
...
}


await 사용법

await는 비동기 처리 코드 앞에 붙인다.

const asyncAPI = async () =>{

<span class="k">await</span> <span class="nx">fetch</span><span class="p">(</span><span class="dl">'</span><span class="s1">https://jayj-fe.github.io/blogAPI/postlist.json</span><span class="dl">'</span><span class="p">)</span>
    <span class="p">.</span><span class="nx">then</span><span class="p">(</span><span class="kd">function</span> <span class="p">(</span><span class="nx">response</span><span class="p">)</span> <span class="p">{</span>
        <span class="nx">console</span><span class="p">.</span><span class="nx">log</span><span class="p">(</span><span class="mi">1</span><span class="p">);</span>
    <span class="p">})</span>

<span class="nx">console</span><span class="p">.</span><span class="nx">log</span><span class="p">(</span><span class="mi">2</span><span class="p">)</span> <span class="p">;</span> 



}


asyncAPI();

결과값 : 1, 2
만약 await 가 없었다면 2, 1 순으로 콘솔에 출력될 것이다.


주의할 점

await가 처리 하고자 하는 비동기 처리 코드는 프로미스를 반환 함수에서만 작동한다.

const asyncAPI = async () =>{

<span class="k">await</span> <span class="nx">setTimeout</span><span class="p">(</span> <span class="p">()</span><span class="o">=&gt;</span><span class="p">{</span> <span class="nx">console</span><span class="p">.</span><span class="nx">log</span><span class="p">(</span> <span class="mi">1</span><span class="p">);</span> <span class="p">}</span>  <span class="p">,</span>  <span class="mi">2000</span> <span class="p">);</span> 

<span class="nx">console</span><span class="p">.</span><span class="nx">log</span><span class="p">(</span><span class="mi">2</span><span class="p">)</span> <span class="p">;</span> 



}


asyncAPI();

결과값이 await를 사용했으므로 1, 2 순으로 콘솔에 출력 될 것을 예상되나,
setTimeout함수는 프로미스를 반환하는 함수가 아니므로 await가 작동하지 않는다.
따라서 결과값은 2, 1 순으로 콘솔창에 출력된다.
