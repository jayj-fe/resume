BEM이란?

BEM은 CSS 네이밍 방법론 중 하나이다.
HTML의 요소를 Block, Element, Modifier 세가지로 구분하여 클래스명을 작명하는 방법이다.


BEM의 장점

  • 목적과 기능을 직관적으로 알 수 있다.
  • 재사용성이 높다.
  • CSS만으로 구조를 알 수 있다.


Block

의미

Block 은 단독으로 사용할 수 있는 독립적인 요소이다.
재사용성이 가능하고 높은 요소이다.
재사용을 위해서 Block 요소에는 Padding과 Margin 속성을 적용하지 않는다.

HTML 구조

*** HTML ***

<header>
<div>
Logo
</div>


&lt;nav&gt;
    Navigater
&lt;/nav&gt;



</header>

위 소스에서 Block은 재사용성이 가능한 요소 Header와 logo의 Div 가 될 수 있겠다.
logo의 경우 footer에서 동일하게 재사용할 수 있다.

CSS Naming

*** CSS ***

<header class="header">
<div class="logo">
Logo
</div>
...
</header>


Element

의미

Element는 블록의 일부이며 블록을 구성하는 종속적인 하위요소이다.
독립적으로 사용할 수 없으며, 독립으로 사용된다면 Element가 아닌 Block이다.
즉, 모든 요소는 블록과 연결된다.

HTML 구조

*** HTML ***

<header>
<div>
Logo
</div>


&lt;nav&gt;
    Navigater
&lt;/nav&gt;



</header>

nav의 메뉴의 경우 Header의 종속적인 요소로 볼 수 있고,
다른 곳에서 재사용이 어렵다고 생각이 들어 Element로 볼 수 있다.

CSS Naming

*** CSS ***

<header class="header">
...
<nav class="header__nav">
Navigater
</nav>
</header>

Element 는 block 이름 + __ ( 두 개의 밑줄 ) + Element 이름 으로 짓는다
Example1 : block __ element
Example2 : header__nav


Modifier

의미

블록이나 엘리먼트의 생김새, 행동, 상태를 바꾸는 데 사용한다.

HTML 구조

*** HTML ***

<header>
<nav>
<a>
Navigater
</a>
</nav>
</header>

CSS Naming

*** CSS ***

<header class="header">
...
<nav class="header__nav">
<a class="header__nav--link-btn">
Navigater
</a>
</nav>
</header>

Modifier 는 Element 이름 + – ( 두 개의 대시 ) + Modifier 이름 으로 짓는다
Example1 : block __ element – modifier
Example2 : header__nav–link-btn
